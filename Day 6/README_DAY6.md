# Day 6: Time Series Forecasting ✈️

## 🎯 Project Overview
Interactive time series analysis dashboard for airline passenger forecasting using ARIMA and Exponential Smoothing models. Features seasonal decomposition, stationarity tests, and multi-month predictions.

## ✨ Key Features
- **6 Comprehensive Tabs:** Data Overview, Time Series Analysis, Stationarity Tests, Model Training, Forecasting, Insights
- **Seasonal Decomposition:** Trend, seasonal, and residual components
- **Stationarity Testing:** ADF test with ACF/PACF plots
- **Dual Models:** ARIMA and Exponential Smoothing comparison
- **Interactive Forecasting:** Customizable prediction periods
- **Performance Metrics:** RMSE, MAE, MAPE evaluation
