# Day 8: Fashion MNIST Classification 👕

## 🎯 Overview
Deep learning image classification using Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN) on Fashion MNIST dataset.

## ✨ Features
- **4 Tabs:** Dataset, Model Training, Evaluation, Predictions
- **2 Architectures:** ANN (Dense layers) and CNN (Convolutional layers)
- **Interactive Training:** Adjustable epochs and batch size
- **10 Classes:** T-shirt, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot
- **Visualizations:** Sample images, confusion matrix, per-class accuracy
- **Live Predictions:** Random test samples with confidence scores

## 🚀 Run
```bash
cd "Day 8"
pip install streamlit plotly tensorflow numpy scikit-learn
streamlit run app_day8.py
```

## 📊 Dataset
**Fashion MNIST:** 60,000 training + 10,000 test grayscale 28x28 images across 10 fashion categories.

## 🎯 Learned
- Neural network architectures (ANN vs CNN)
- Image preprocessing and normalization
- Conv2D, MaxPooling, Dropout layers
- Model training and evaluation
- Confusion matrix analysis
