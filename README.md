## 🎯 Learning Journey

### 📊 **Traditional Machine Learning** (Days 1-7)
- **Exploratory Data Analysis**: Titanic & Netflix datasets
- **Regression**: House price prediction with 5 ML models
- **Classification**: Heart disease prediction, churn analysis
- **Clustering**: Customer segmentation (K-Means, Hierarchical)
- **Time Series**: Airline passenger forecasting (ARIMA, Exponential Smoothing)

### 🧠 **Deep Learning** (Days 8-10)
- **Neural Networks**: Fashion MNIST (ANN & CNN)
- **Transfer Learning**: Pre-trained models (ResNet50, VGG16, MobileNetV2)
- **Generative Models**: GANs for digit generation

### 🤖 **Modern AI & LLMs** (Days 11-14)
- **HuggingFace**: Sentiment analysis, summarization, Q&A
- **Computer Vision**: Object detection, image enhancement
- **LSTM Networks**: Stock price prediction with technical indicators
- **GPT Architecture**: Understanding and building transformers

### 🌟 **Advanced GenAI** (Days 15-21)
- **NL to SQL**: Natural language database queries
- **OCR Automation**: Resume parsing, entity extraction
- **Search Engine**: Web crawler, TF-IDF, PageRank
- **RAG System**: Vector similarity, knowledge base chatbot
- **AI Agents**: Multi-agent collaboration (CrewAI)
- **Browser Automation**: Playwright/Selenium for web scraping
- **Newsletter Pipeline**: End-to-end automation with n8n

## 🚀 Quick Start

### Prerequisites
```bash
Python 3.8+
pip (Python package manager)
```

### For Contributors: Upload Options

**Option 1: Upload All at Once** (See `QUICK_COMMANDS.txt`)
- Fast setup - all 21 projects uploaded immediately

**Option 2: Gradual Upload - 21 Days Challenge** (See `GRADUAL_UPLOAD_GUIDE.md`)
- Upload one project per day over 21 days
- Build a GitHub streak and daily visibility
- Recommended for personal branding

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/21-days-ml-dl-genai.git
cd 21-days-ml-dl-genai
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Run any day's application**
```bash
# Example: Run Day 1 (Titanic EDA)
cd "Day 1"
streamlit run app_day1.py

# Example: Run Day 18 (RAG Chatbot)
cd "Day 18"
streamlit run app_day18.py
```

4. **Access in browser**
```
http://localhost:8501
```
