# Day 7: Customer Churn Prediction 📱

## 🎯 Overview
Telecom customer churn prediction using feature engineering and machine learning classification models.

## ✨ Features
- **4 Tabs:** Overview, Analysis, Models, Insights
- **Feature Engineering:** Tenure groups, total services, average spending
- **3 ML Models:** Logistic Regression, Random Forest, Gradient Boosting
- **Metrics:** Accuracy, Precision, Recall, F1, ROC-AUC
- **Visualizations:** Churn distribution, contract analysis, ROC curves

## 🚀 Run
```bash
cd "Day 7"
pip install streamlit plotly pandas scikit-learn
streamlit run app_day7.py
```

## 📊 Dataset
**Telco Customer Churn:** 7,043 customers with 21 features including tenure, charges, services, and contract type.

## 🎯 Learned
- Feature engineering techniques
- Churn analysis and patterns
- Classification model comparison
- Business metrics and recommendations
